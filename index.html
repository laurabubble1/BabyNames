French Baby Names: Interactive Visualizations

Explore trends in baby names in France from 1900 to 2019. Use the controls below each visualization to interact with the data.

1. Name Popularity Over Time

Strengths: Allows comparison of multiple names over time, reveals trends and sudden popularity changes.
Weaknesses: Can become cluttered with too many names; requires user to know which names to search.

2. Regional Name Popularity

Strengths: Shows geographic distribution of name popularity; easy to spot regional effects.
Weaknesses: May not show trends over time ; slows down with many names;

3. Gender Differences in Name Popularity

Strengths: Compares popularity by gender over time; highlights unisex names.
Weaknesses: Only for names used by both genders; may be less relevant for strictly gendered names.
